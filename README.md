# React Quickstart Template ⚡
React Typescript template project to quickly bootstrap an idea ⚡

## Project Setup 🚀
`yarn install`

## Available Scripts 📜 
| Command   | Description            |
|-----------|------------------------|
| start     | development server     |
| build     | builds for production  |


## Packages 🗳️
- **UI / CSS / Animation**
  - Tailwind CSS
  - Tailwind CSS Typography
  - Chakra UI
  - Emotion React
  - Framer Motion
  - PostCSS
- **STATE**
  - Immer
- **Router**
  - Reach Router
- **MISC**
  - React Query
